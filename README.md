# Dükkan Uygulaması

- Kayıtlı olan müşterilerin uygulamaya giriş yaptıktan sonra mağazadaki ürün listesini görüntüyebildiği ve ürüne tıklayarak içerik ve fiyat bilgini görüntüleyebildiği bir uygulama.
